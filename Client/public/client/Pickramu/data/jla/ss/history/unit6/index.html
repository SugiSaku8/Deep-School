Unit 6: 世界史の流れ

第1章: 古代文明と帝国

🏛️ 学習の目標

このUnitでは、世界史の大きな流れについて学びます。
古代文明から現代まで、世界の歴史を理解しましょう！

📖 資料の提示

四大文明
• エジプト文明（ナイル川流域）
• メソポタミア文明（チグリス・ユーフラテス川）
• インダス文明（インダス川流域）
• 黄河文明（黄河流域）
古代帝国
• アケメネス朝ペルシア
• アレクサンドロス帝国
• ローマ帝国
• 漢帝国

🎯 要点

文明の特徴：
• 大河の流域で発展
• 農業の発達
• 文字の発明
• 都市の形成

帝国の特徴：
• 広大な領土
• 中央集権的な統治
• 多民族の統合
• 文化の交流

📝 要点のまとめ

古代文明の発展
大河 → 農業 → 都市 → 文明 → 帝国

文明の共通点
文字・都市・農業・統治システム

古代文明は、大河の流域で農業が発達し、都市が形成されて文明が生まれました。

ピラミッド
エジプトの王（ファラオ）の墓。
紀元前2600年頃に建造。
世界七不思議の一つ。
シルクロード
中国とヨーロッパを結ぶ交易路。

文化・宗教も伝播した。

📖 資料の提示（中世・近世）

中世ヨーロッパ
• 封建制度の成立
• キリスト教の影響
• 十字軍遠征
• 都市の発達
大航海時代
• スペイン・ポルトガルの進出
• 新大陸の発見
• 世界貿易の開始
• 植民地の形成

🎯 要点（中世・近世）

封建制度：
• 領主と家臣の関係
• 自給自足の経済
• 教会の権威
• 身分制度

大航海時代の影響：
• 世界市場の形成
• ヨーロッパの優位
• 植民地支配の開始
• 文化の交流

📝 要点のまとめ（中世・近世）

中世から近世へ
封建制度 → 都市発達 → 大航海 → 世界市場

変化の要因
技術革新・経済発展・宗教改革・科学革命

中世から近世にかけて、ヨーロッパは世界の中心として台頭しました。

💡 豆知識（中世・近世）

十字軍
11-13世紀、キリスト教徒による聖地奪回運動。
8回の遠征が行われた。
東西文化の交流をもたらした。
ルネサンス
14-16世紀のヨーロッパ文化運動。
古代ギリシャ・ローマ文化の復興。
人文主義の思想が広がった。
