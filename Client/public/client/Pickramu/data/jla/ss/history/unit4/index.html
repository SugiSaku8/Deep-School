Unit 4: 近代の日本と世界

第1章: 開国と明治維新

📚 学習の目標

このUnitでは、近代日本の成立と発展について学びます。
開国から明治維新、そして近代国家への道筋を理解しましょう！

📖 資料の提示

開国の背景
• 1853年：ペリー来航
• 1854年：日米和親条約
• 1858年：日米修好通商条約
• 不平等条約の締結
明治維新の流れ
• 1867年：大政奉還
• 1868年：明治維新
• 廃藩置県・地租改正
• 富国強兵・殖産興業

🎯 要点

開国の影響：
• 鎖国体制の崩壊
• 不平等条約の締結
• 攘夷運動の高まり
• 幕府の権威低下

明治維新の改革：
• 中央集権国家の建設
• 身分制度の廃止
• 近代産業の育成
• 軍備の強化

📝 要点のまとめ

開国から明治維新へ
ペリー来航 → 不平等条約 → 攘夷運動 → 大政奉還 → 明治維新

明治政府の政策
廃藩置県・地租改正・富国強兵・殖産興業

開国は日本の近代化への第一歩であり、明治維新により近代国家が建設されました。

💡 豆知識

黒船来航
1853年、ペリーが4隻の軍艦で浦賀に来航。
蒸気船の黒い船体から「黒船」と呼ばれた。
日本に開国を迫る圧力となった。
五箇条の御誓文
1868年、明治天皇が示した新政府の基本方針。
「広く会議を興し、万機公論に決すべし」
民主主義の萌芽を示す文書。

問題2: 明治維新が日本の近代化に果たした役割を、政治・経済・社会の変化から説明しなさい。

開国と明治維新を理解するコツ

2. 国内の政治状況を把握する
3. 改革の目的と効果を考える

開国と明治維新は、日本が近代国家として発展するための重要な転換点でした。

📖 資料の提示（日清・日露戦争）

日清戦争（1894-1895年）
• 朝鮮半島の支配権をめぐる戦争
• 日本が勝利
• 下関条約の締結
• 台湾・澎湖諸島の割譲
日露戦争（1904-1905年）
• 満州・朝鮮の支配権をめぐる戦争
• 日本が勝利
• ポーツマス条約の締結
• 南樺太の割譲

🎯 要点（日清・日露戦争）

戦争の背景：
• 帝国主義の時代
• 列強のアジア進出
• 日本の軍国主義化
• 朝鮮半島の重要性

戦争の影響：
• 日本の国際的地位向上
• 軍備拡張の加速
• 国民の戦争熱
• アジアへの影響

📝 要点のまとめ（日清・日露戦争）

戦争の流れ
帝国主義時代 → 朝鮮半島争奪 → 日清戦争 → 日露戦争 → 国際的地位向上

日清・日露戦争は、日本が帝国主義国家として台頭する契機となりました。

💡 豆知識（日清・日露戦争）

黄海海戦
日清戦争中の海戦。日本海軍が清国北洋艦隊に勝利。
日本の海軍力が世界に認められる契機。
近代海軍の重要性を示した。
旅順攻囲戦
日露戦争中の陸戦。日本軍が旅順要塞を攻略。
約6ヶ月の長期戦。
近代戦の過酷さを示した。
