Unit 1: 古代までの日本

第1章: 原始・古代の日本

📚 学習の目標

このUnitでは、日本の原始・古代の歴史について学びます。
縄文時代から古墳時代まで、日本の古代社会の成り立ちを理解しましょう！

📖 資料の提示

縄文土器の特徴
• 縄目の文様が特徴的
• 深鉢形の土器が多い
• 煮炊きに使用
• 約1万年前〜紀元前4世紀頃
弥生土器の特徴
• 薄手で硬い
• 赤褐色で光沢がある
• 甕（かめ）や高坏（たかつき）が特徴
• 紀元前4世紀頃〜紀元3世紀頃

🎯 要点

縄文時代の特徴：
• 狩猟・採集生活
• 竪穴住居に居住
• 貝塚が形成される
• 土偶や石棒などの祭祀具

弥生時代の特徴：
• 稲作の開始
• 金属器の使用
• 環濠集落の出現
• 貧富の差の発生

📝 要点のまとめ

縄文時代
狩猟・採集生活 → 竪穴住居・貝塚 → 土偶・石棒

弥生時代
稲作開始 → 金属器使用 → 環濠集落 → 貧富の差

縄文時代は自然と共生する生活、弥生時代は農耕社会への移行期でした。

💡 豆知識

三内丸山遺跡
青森県にある縄文時代の大規模集落遺跡。
長期間（約1500年間）にわたって継続的に営まれた。
大型の掘立柱建物や、栗の栽培跡が発見されている。
吉野ヶ里遺跡
佐賀県にある弥生時代の環濠集落。
物見櫓や柵列、倉庫群が復元されている。
弥生時代の社会構造を理解する重要な遺跡。

❓ 発展問題

問題1: 縄文時代と弥生時代の生活の違いを、住居・食生活・社会構造の3つの観点から説明しなさい。

問題2: 弥生時代に貧富の差が生まれた理由を、稲作の特徴から説明しなさい。

🔑 発展のコツ

時代の特徴を比較するコツ
1. 衣食住の変化を押さえる
2. 技術の進歩に注目する
3. 社会構造の変化を理解する
4. 遺跡や出土品から当時の生活を想像する

縄文→弥生の変化は、日本が農耕社会へ移行する重要な転換期でした。

📖 資料の提示（古墳時代）

前方後円墳の特徴
• 円形と方形を組み合わせた墳丘
• 3世紀後半〜7世紀頃に築造
• 権力者の墓として使用
• 埴輪や副葬品を伴う
古墳時代の社会
• ヤマト政権の成立
• 豪族による支配
• 朝鮮半島との交流
• 仏教の伝来

🎯 要点（古墳時代）

古墳時代の特徴：
• 前方後円墳の築造
• ヤマト政権の成立
• 豪族による支配体制
• 朝鮮半島との活発な交流
• 仏教の伝来（538年）

ヤマト政権：
• 大和地方を中心とした政治勢力
• 豪族の連合政権
• 大王（おおきみ）を中心とした統治

📝 要点のまとめ（古墳時代）

古墳時代の流れ
前方後円墳築造 → ヤマト政権成立 → 豪族支配 → 朝鮮半島交流 → 仏教伝来

古墳時代は、日本が統一政権を形成し、大陸との交流を深めた時代でした。

💡 豆知識（古墳時代）

仁徳天皇陵古墳
大阪府堺市にある日本最大の前方後円墳。
全長486m、高さ35m。
世界文化遺産に登録されている。
埴輪（はにわ）
古墳に立て並べられた土製品。
人物・動物・家屋・器物の形をしている。
死者を守る役割や、権威の象徴として使用された。

問題3: 前方後円墳が権力者の墓として選ばれた理由を、その形状の特徴から説明しなさい。

問題4: 古墳時代に朝鮮半島との交流が活発になった理由を、当時の東アジア情勢から説明しなさい。

🔑 発展のコツ（古墳時代）

古墳時代を理解するコツ
1. 古墳の規模から権力の大きさを読み取る
2. 副葬品から当時の技術水準を理解する
3. 朝鮮半島との交流から国際関係を把握する
4. 仏教伝来の影響を考える

古墳時代は、日本が統一国家への道を歩み始めた重要な時代でした。
