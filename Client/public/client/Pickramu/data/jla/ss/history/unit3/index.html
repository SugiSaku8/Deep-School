Unit 3: 近世の日本

第1章: 戦国時代と織豊政権

📚 学習の目標

このUnitでは、近世日本の政治・社会・文化について学びます。
戦国時代から江戸時代まで、統一政権の成立と社会の変化を理解しましょう！

📖 資料の提示

戦国時代の特徴
• 応仁の乱（1467年）から始まる
• 下克上の風潮
• 戦国大名の台頭
• 城下町の発達
織豊政権の成立
• 織田信長：天下統一の基礎を築く
• 豊臣秀吉：全国統一を完成
• 太閤検地・刀狩り
• 朝鮮出兵（文禄・慶長の役）

🎯 要点

戦国大名の特徴：
• 領国経営を行う
• 家臣団を組織化
• 城下町を建設
• 経済基盤を確立

織田信長の政策：
• 楽市・楽座の実施
• キリスト教を容認
• 比叡山延暦寺を焼き討ち
• 安土城を築城

📝 要点のまとめ

戦国時代から統一へ
応仁の乱 → 戦国大名台頭 → 織田信長 → 豊臣秀吉 → 全国統一

統一政策
太閤検地・刀狩り・城下町建設・経済政策

戦国時代は、下克上の時代でありながら、全国統一への道筋が作られた時代でした。

💡 豆知識

安土城
織田信長が築いた城。天守閣を持つ本格的な城郭。
楽市・楽座により商業が発展。
本能寺の変後に焼失した。
太閤検地
豊臣秀吉が実施した全国的な土地調査。
石高制の確立。
荘園制度の完全な解体。

❓ 発展問題

問題1: 戦国大名が城下町を建設した理由を、軍事・経済の観点から説明しなさい。

問題2: 太閤検地が荘園制度の解体につながった理由を、土地制度の変化から説明しなさい。

🔑 発展のコツ

戦国時代を理解するコツ
1. 下克上の意味を理解する
2. 戦国大名の政策を押さえる
3. 統一への流れを把握する
4. 経済政策の影響を考える

戦国時代は、混乱の時代でありながら、新しい社会秩序が生まれる転換期でした。

📖 資料の提示（江戸時代）

江戸幕府の成立
• 1603年：徳川家康が征夷大将軍に任命
• 幕藩体制の確立
• 参勤交代制度
• 鎖国政策
江戸時代の社会
• 士農工商の身分制度
• 村請制による年貢徴収
• 都市の発達
• 元禄文化の繁栄

🎯 要点（江戸時代）

幕藩体制：
• 将軍と大名の主従関係
• 大名の領地支配
• 参勤交代による統制
• 武家諸法度による規制

鎖国政策：
• 1639年：ポルトガル船の来航禁止
• オランダ・中国・朝鮮のみと貿易
• 長崎出島での貿易
• キリスト教の禁止

📝 要点のまとめ（江戸時代）

江戸時代の政治体制
幕府成立 → 幕藩体制 → 参勤交代 → 鎖国政策 → 身分制度

江戸時代は、徳川幕府による安定した統治体制が確立された時代でした。

💡 豆知識（江戸時代）

元禄文化
17世紀末から18世紀初頭の文化。
井原西鶴・松尾芭蕉・近松門左衛門が活躍。
町人文化が発展した。
参勤交代
大名が1年おきに江戸と国元を往復する制度。
大名の経済力を弱める効果。
江戸の都市化を促進した。

問題3: 参勤交代が大名の経済力を弱める理由を、費用と効果から説明しなさい。

問題4: 鎖国政策が日本の発展に与えた影響を、技術・文化の観点から説明しなさい。

🔑 発展のコツ（江戸時代）

江戸時代を理解するコツ
1. 幕藩体制の仕組みを理解する
2. 鎖国政策の目的を考える
3. 身分制度の特徴を押さえる
4. 文化の発展を評価する

江戸時代は、日本独自の政治・社会制度が確立され、安定した統治が行われた時代でした。
