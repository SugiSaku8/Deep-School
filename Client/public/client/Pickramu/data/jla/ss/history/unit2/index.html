Unit 2: 中世の日本

第1章: 平安時代の政治と文化

📚 学習の目標

このUnitでは、中世日本の政治・社会・文化について学びます。
平安時代から戦国時代まで、武士の台頭と社会の変化を理解しましょう！

📖 資料の提示

平安時代の政治制度
• 摂関政治（藤原氏による外戚政治）
• 荘園制度の発達
• 国司制度の崩壊
• 武士の台頭
平安時代の文化
• 国風文化（和風文化）
• 仮名文字の発達
• 貴族文化の最盛期
• 浄土教の流行

🎯 要点

摂関政治の特徴：
• 藤原氏が天皇の外戚として権力を握る
• 摂政・関白の職を独占
• 娘を天皇の后として送り込む
• 政治の実権を掌握

荘園制度：
• 貴族や寺社が私有地を持つ
• 国司の支配を逃れる
• 年貢を納めない特権を持つ
• 武士が荘官として管理

📝 要点のまとめ

平安時代の政治構造
摂関政治 → 荘園制度発達 → 国司制度崩壊 → 武士台頭

文化の特徴
国風文化 → 仮名文字 → 貴族文化 → 浄土教

平安時代は、貴族政治の最盛期でありながら、武士勢力が台頭する転換期でした。

💡 豆知識

紫式部と清少納言
紫式部：『源氏物語』の作者。宮廷の恋愛を描いた長編小説。
清少納言：『枕草子』の作者。随筆文学の傑作。
両者とも平安時代の女流文学者として有名。
浄土教
阿弥陀仏の極楽浄土に往生することを願う仏教。
末法思想と結びついて流行。
平等院鳳凰堂は浄土教の影響を受けて建立された。

平安時代は、貴族文化の最盛期でありながら、武士社会への転換期でもありました。

📖 資料の提示（鎌倉時代）

鎌倉幕府の成立
• 1192年：源頼朝が征夷大将軍に任命
• 御家人制度の確立
• 守護・地頭の設置
• 武家社会の確立
鎌倉時代の政治
• 執権政治（北条氏）
• 承久の乱（1221年）
• 元寇（1274年・1281年）
• 御家人の窮乏化

🎯 要点（鎌倉時代）

御家人制度：
• 主従関係による武士団の組織化
• 御恩と奉公の関係
• 所領の安堵と新恩給与
• 軍役の義務

執権政治：
• 北条氏が執権として実権を握る
• 得宗専制政治
• 御家人の不満が高まる
• 幕府の衰退

📝 要点のまとめ（鎌倉時代）

鎌倉時代の流れ
幕府成立 → 御家人制度 → 執権政治 → 承久の乱 → 元寇 → 幕府衰退

鎌倉時代は、武家社会が確立され、日本史上初めて武士が政権を握った時代でした。

💡 豆知識（鎌倉時代）

承久の乱
1221年、後鳥羽上皇が北条義時追討の院宣を発した。
幕府軍が勝利し、上皇は隠岐に流された。
朝廷の権威が失墜し、武家政治が確立された。
元寇
1274年（文永の役）と1281年（弘安の役）の2回。
元・高麗連合軍が日本に侵攻。
神風（台風）により撃退された。
