Unit 3: 経済と私たちの生活

第1章: 市場経済の仕組み

💰 学習の目標

このUnitでは、経済の仕組みと私たちの生活について学びます。
市場経済・金融・国際経済の基本を理解しましょう！

📖 資料の提示

市場経済の基本原理
• 需要と供給の法則
• 価格の自動調整機能
• 競争による効率化
• 自由な取引
経済主体
• 家計（消費者）
• 企業（生産者）
• 政府（公共部門）
• 海外（外国）

🎯 要点

需要と供給の関係：
• 価格が上がると需要が減り、供給が増える
• 価格が下がると需要が増え、供給が減る
• 需要と供給が一致する点で価格が決まる
• 市場の自動調整機能

経済循環：
• 家計から企業への労働力提供
• 企業から家計への賃金支払い
• 家計から企業への商品購入
• 企業から家計への商品提供

📝 要点のまとめ

市場経済の仕組み
需要と供給 → 価格決定 → 自動調整 → 効率的配分

経済循環
家計・企業・政府・海外の相互依存関係

市場経済は、需要と供給の関係により価格が決まり、資源が効率的に配分される仕組みです。

💡 豆知識

見えざる手
アダム・スミスが提唱した市場経済の原理。
個人の利益追求が社会全体の利益につながる。
市場の自動調整機能を表現した言葉。
GDP（国内総生産）
一国で1年間に生産された財・サービスの総額。
経済規模を測る重要な指標。
日本は世界第3位の経済大国。

❓ 発展問題

問題1: 市場経済が効率的な資源配分を実現する理由を、需要と供給の関係から説明しなさい。

問題2: 政府が市場経済に介入する理由を、市場の失敗と公共財の観点から説明しなさい。

🔑 発展のコツ

経済を理解するコツ
1. 需要と供給の関係を理解する
2. 経済主体の役割を把握する
3. 市場の機能と限界を考える
4. 政府の役割を理解する

経済の仕組みを理解することで、私たちの生活と経済の関係が見えてきます。

📖 資料の提示（金融と国際経済）

金融の役割
• 資金の仲介機能
• リスク分散機能
• 決済機能
• 情報提供機能
国際経済の特徴
• グローバル化の進展
• 国際分業の深化
• 為替レートの変動
• 国際協力の重要性

🎯 要点（金融と国際経済）

金融機関の種類：
• 中央銀行（日本銀行）
• 民間銀行
• 証券会社
• 保険会社

国際経済の課題：
• 経済格差の拡大
• 環境問題の深刻化
• 通貨危機の発生
• 貿易摩擦の増加

📝 要点のまとめ（金融と国際経済）

金融の機能
資金仲介・リスク分散・決済・情報提供

国際経済
グローバル化・国際分業・為替変動・国際協力

金融は経済活動を支える重要な機能を持ち、国際経済はグローバル化が進んでいます。

💡 豆知識（金融と国際経済）

日本銀行
日本の中央銀行。通貨発行・金融政策を担当。
物価の安定と金融システムの安定を目指す。
独立した機関として運営されている。
WTO（世界貿易機関）
国際貿易のルールを定める国際機関。
自由貿易の促進と紛争解決を担当。
164か国・地域が加盟している。

問題3: 金融機関が経済発展に果たす役割を、資金の流れから説明しなさい。

問題4: グローバル化が進む中で、日本が取るべき経済政策を、国際競争力の観点から説明しなさい。

🔑 発展のコツ（金融と国際経済）

金融と国際経済を理解するコツ
1. 金融の基本機能を理解する
2. 国際経済の特徴を把握する
3. グローバル化の影響を考える
4. 国際協力の重要性を認識する

金融と国際経済は、現代社会の経済活動を支える重要な要素です。
