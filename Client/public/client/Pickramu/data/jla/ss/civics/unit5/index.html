Unit 5: 地方自治と私たち

第1章: 地方自治の仕組み

🏛️ 学習の目標

このUnitでは、地方自治の仕組みについて学びます。
地方自治体・住民参加・地域づくりを理解しましょう！

📖 資料の提示

地方自治体の種類
• 都道府県（47都道府県）
• 市町村（約1700市町村）
• 特別区（東京都23区）
• 政令指定都市（20市）
地方自治体の仕事
• 住民福祉の向上
• 地域の安全確保
• 教育・文化の振興
• 産業の発展

🎯 要点

地方自治の原則：
• 住民自治（住民が主体）
• 団体自治（地方自治体の自主性）
• 補完性の原則（身近な行政）
• 住民参加の促進

地方自治体の機関：
• 議会（議決機関）
• 首長（執行機関）
• 委員会（専門機関）
• 監査委員（監査機関）

📝 要点のまとめ

地方自治の仕組み
住民自治・団体自治・補完性・住民参加

自治体の機関
議会・首長・委員会・監査委員

地方自治は、住民が主体となって地域の課題を解決する仕組みです。

💡 豆知識

地方自治法
1947年に制定された地方自治の基本法。
地方自治体の組織・運営を規定。
住民の権利と義務を定めている。
住民投票
重要な政策について住民の意思を問う制度。
条例制定・改廃の請求権。
直接民主主義の一形態。

❓ 発展問題

問題1: 地方自治が民主主義に果たす役割を、住民参加と意思決定の観点から説明しなさい。

問題2: 都道府県と市町村の役割分担を、行政の効率性の観点から説明しなさい。

🔑 発展のコツ

地方自治を理解するコツ
1. 住民自治の原則を理解する
2. 機関の役割を把握する
3. 住民参加の重要性を認識する
4. 地域の課題を考える

地方自治は、私たちの生活に最も身近な民主主義の実践です。

📖 資料の提示（住民参加と地域づくり）

住民参加の方法
• 住民投票
• パブリックコメント
• 住民説明会
• 審議会・委員会
地域づくりの課題
• 人口減少・高齢化
• 地域経済の衰退
• 環境問題
• 防災・減災

🎯 要点（住民参加と地域づくり）

住民参加の意義：
• 民主主義の深化
• 政策の透明性向上
• 住民の満足度向上
• 地域の活性化

地域づくりの取り組み：
• まちづくり条例の制定
• 地域計画の策定
• 市民活動の支援
• 協働の推進

📝 要点のまとめ（住民参加と地域づくり）

住民参加の方法
住民投票・パブリックコメント・説明会・審議会

地域づくりの課題
人口減少・経済衰退・環境問題・防災減災

住民参加は、地域づくりを成功させるための重要な要素です。

💡 豆知識（住民参加と地域づくり）

パブリックコメント
政策案について住民の意見を求める制度。
政策決定の透明性を高める。
住民の声を政策に反映させる。
協働
行政と住民が協力して課題を解決する取り組み。
行政の効率性と住民の主体性を両立。
新しい公共の概念。

問題3: 住民参加が地域づくりに果たす役割を、民主主義と政策効果の観点から説明しなさい。

問題4: 私たちが地域づくりに参加する方法を、日常生活の観点から説明しなさい。

🔑 発展のコツ（住民参加と地域づくり）

住民参加と地域づくりを理解するコツ

2. 地域の課題を認識する
3. 協働の重要性を考える

地域づくりは、住民一人一人の参加によって支えられています。
